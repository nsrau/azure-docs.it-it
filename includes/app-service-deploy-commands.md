* [Gestire le risorse di Azure tramite Azure PowerShell](../articles/azure-resource-manager/management/manage-resources-powershell.md)
* [Gestire le risorse di Azure tramite l'interfaccia della riga di comando di Azure](../articles/azure-resource-manager/management/manage-resources-cli.md).
