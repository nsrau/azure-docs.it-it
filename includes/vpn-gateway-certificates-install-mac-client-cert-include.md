Quando si installa un certificato client, è necessaria la password che è stata creata durante l'esportazione del certificato client.

1. Trovare il file di certificato PFX e copiarlo nel Mac. È possibile trasferire il certificato nel Mac in diversi modi, ad esempio inviando il file tramite posta elettronica.
2. Dopo averlo copiato nel Mac, fare doppio clic sul certificato per aprire la finestra **Aggiungi certificati** e fare clic su **Aggiungi** per avviare l'installazione.

  ![Aggiungere certificati](./media/vpn-gateway-certificates-install-mac-client-cert-include/addcert.png)
3. Immettere la password creata durante l'esportazione del certificato client. La password consente di proteggere la chiave privata del certificato. Fare clic su **OK** per completare l'installazione.

  ![Password](./media/vpn-gateway-certificates-install-mac-client-cert-include/password.png)
