- **[Monitorare con dashboard Azure Data Studio](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> Azure Data Studio è uno strumento di database multipiattaforma per professionisti della gestione di dati che usano la famiglia Microsoft di piattaforme dati locali e cloud in Windows, MacOS e Linux. Azure Data Studio offre un'esperienza di modifica moderna per la gestione dei dati tra più origini con funzionalità IntelliSense, frammenti di codice, integrazione del controllo del codice sorgente e un terminale integrato. Azure Data Studio è stato progettato in base alle esigenze dell'utente della piattaforma dati, con grafici predefiniti di set di risultati delle query e dashboard personalizzabili.

**[Monitorare con Grafana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana è un'applicazione Web per la visualizzazione interattiva e l'analisi open source multipiattaforma. Fornisce grafici, grafici e avvisi per il Web quando si è connessi a origini dati supportate. È espandibile tramite un sistema di plug-in. Gli utenti finali possono creare dashboard di monitoraggio complessi tramite query interattive.

**[Log Analytics con Kibana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana è un dashboard di visualizzazione dei dati open source per ElasticSearch. Fornisce funzionalità di visualizzazione oltre al contenuto indicizzato in un cluster ElasticSearch. Gli utenti possono creare grafici a barre, a linee e a dispersione oppure grafici a torta e mappe su grandi volumi di dati.
