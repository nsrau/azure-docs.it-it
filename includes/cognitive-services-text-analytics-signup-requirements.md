Una chiave e un endpoint per una risorsa di Analisi del testo. I Servizi cognitivi di Azure sono rappresentati dalle risorse di Azure a cui si effettua la sottoscrizione. Creare una risorsa per Analisi del testo usando il [portale di Azure](../articles/cognitive-services/cognitive-services-apis-create-account.md) o l'[interfaccia della riga di comando di Azure](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) nel computer locale. È anche possibile:

* Ottenere un [codice di valutazione gratuito](https://azure.microsoft.com/try/cognitive-services/#lang) valido per 7 giorni. Dopo aver eseguito l'iscrizione, sarà disponibile sul [sito Web di Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/).

* Visualizzare questa risorsa nel [portale di Azure](https://portal.azure.com/).
