Per altre informazioni sugli SKU di gateway di versione precedente, vedere [Working with virtual network gateway SKUs (old)](../articles/vpn-gateway/vpn-gateway-about-skus-legacy.md) (Uso degli SKU di gateway di rete virtuale - Versione precedente).
