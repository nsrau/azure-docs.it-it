## <a name="create-an-azure-storage-account"></a>Creare un account di archiviazione di Azure
Il modo più semplice per creare il primo account di archiviazione di Azure consiste nell'usare il [portale di Azure](https://portal.azure.com). Per altre informazioni, vedere [Creare un account di archiviazione](https://docs.microsoft.com/azure/storage/common/storage-quickstart-create-account).

È anche possibile creare un account di archiviazione di Azure usando [Azure PowerShell](../articles/storage/common/storage-powershell-guide-full.md), l'[interfaccia della riga di comando di Azure](../articles/storage/common/storage-azure-cli.md) o la [libreria client del provider di risorse di archiviazione per .NET](https://msdn.microsoft.com/library/azure/mt131037.aspx).
