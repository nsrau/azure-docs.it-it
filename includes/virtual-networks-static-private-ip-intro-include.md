A una macchina virtuale (VM) viene assegnato automaticamente un indirizzo IP privato da un intervallo specificato, in base alla subnet in cui è distribuita la macchina virtuale. La macchina virtuale mantiene l'indirizzo finché la macchina virtuale non viene eliminata. Azure assegna dinamicamente il successivo indirizzo IP privato disponibile dalla subnet in cui si crea una macchina virtuale. Se si vuole un indirizzo IP specifico dalla subnet assegnata alla macchina virtuale, assegnare un indirizzo IP statico.
