## <a name="overview-azure-database-for-postgresql-and-mysql"></a>Overview: Azure Database for PostgreSQL and MySQL (Panoramica: Database di Azure per PostgreSQL e Database di Azure per MySQL)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

I servizi Database di Azure per PostgreSQL e Database di Azure per MySQL combinano i motori di database e le funzionalità di tipo Community Edition di un servizio completamente gestito in modo da consentire di concentrarsi sulla creazione di app anziché sulla gestione del database. In questo video viene presentata una panoramica dei vantaggi derivanti dall'uso del sevizio e delle principali funzionalità disponibili.

## <a name="create-a-postgresql-or-mysql-server"></a>Creare un server PostgreSQL o MySQL
Database di Azure per PostgreSQL e database di Azure per MySQL sono servizi gestiti usati per eseguire, gestire e ridimensionare i motori di database di Community Edition a disponibilità elevata nel cloud. In queste esercitazioni video viene illustrato come creare un server PostgreSQL o MySQL in circa tre minuti usando il portale di Azure. Se non si ha una sottoscrizione di Azure, [creare un account Azure gratuito](https://azure.microsoft.com/free/) prima di iniziare.

* [Esercitazione video PostgreSQL](https://azure.microsoft.com/resources/videos/create-an-azure-database-for-postgresql-server-in-the-azure-portal)

* [Esercitazione video su MySQL](https://azure.microsoft.com/resources/videos/create-an-azure-database-for-mysql-server-by-using-the-azure-portal)
