# <a name="get-started-with-the-excel-connector"></a>Introduzione al connettore Excel
Non sono disponibili connettori Excel nelle app per la logica. 

## <a name="to-use-excel-data"></a>Per usare dati di Excel
È possibile archiviare dati di Excel come file con valori delimitati da virgole (CSV) in una cartella di archiviazione, ad esempio [OneDrive](connectors-create-api-onedrive.md). Questo file CSV può essere usato anche con il [connettore file flat](../logic-apps/logic-apps-enterprise-integration-flatfile.md).

## <a name="connector-specific-details"></a>Dettagli specifici del connettore

Per visualizzare eventuali azioni e trigger definiti in Swagger ed eventuali limiti, vedere i [dettagli del connettore](/connectors/excel/).
