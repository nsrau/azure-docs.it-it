# <a name="setting-up-access-for-azure-dev-tools"></a>Configurazione dell'accesso per Azure Dev Tools

Per accedere alla sottoscrizione in modo da distribuire il software agli studenti e organizzare i lab, sono disponibili due modi:
1. Scaricare il software e le chiavi dal portale Sottoscrizione di Visual Studio.
1. Concedere l'accesso degli studenti allo store dell'hub di Education.

Per altre informazioni, visitare il [sito Web Azure for Education](https://azureforeducation.microsoft.com).

## <a name="for-administrators-only"></a>Solo per amministratori  
Gli amministratori accedono agli strumenti di sviluppo di Azure tramite il [portale Sottoscrizione di Visual Studio](https://my.visualstudio.com/).

Per richiedere l'accesso a Sottoscrizione di Visual Studio, è possibile usare il [portale di gestione](https://azureforeducation.microsoft.com/account/Subscriptions). Una volta ricevuta l'approvazione, si avrà accesso al portale Sottoscrizione di Visual Studio. La prima volta che si accede al portale Sottoscrizione di Visual Studio, verrà chiesto di collegare un account Microsoft all'indirizzo di posta elettronica specificato al momento della registrazione. Per collegare questi account, sarà necessario fornire l'ID sottoscrittore o il numero di accesso ai vantaggi, il nome e il cognome di un amministratore e l'indirizzo di posta elettronica specificato durante la registrazione. Dopo aver collegato gli account, sarà possibile accedere al portale Sottoscrizioni di Visual Studio.

## <a name="for-students-faculty-and-administrators"></a>Per studenti, docenti e amministratori
Gli studenti accedono agli strumenti di sviluppo tramite l'[hub di Education](https://aka.ms/devtoolsforteaching).

Studenti e docenti possono accedere a tutti i download di software tramite l'hub di Education. L'hub di Education è integrato nel portale di Azure e fornisce agli studenti l'accesso semplificato all'intero catalogo di software, oltre all'accesso all'intero catalogo di [Microsoft Learn](/learn/).
