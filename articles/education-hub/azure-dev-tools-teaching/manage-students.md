# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>Registrazione di studenti in Azure Dev Tools for Teaching
Dopo l'acquisto di Azure Dev Tools for Teaching, tutti gli studenti che si trovano nel dominio dell'istituzione otterranno automaticamente l'accesso a questa offerta. Durante il processo di acquisto, verrà chiesto di immettere l'indirizzo di posta elettronica dell'amministratore della sottoscrizione. Questo dominio verificato dall'università sbloccherà gli studenti con lo stesso dominio di posta elettronica e concederà loro la facoltà di registrarsi e riscattare i vantaggi offerti dal download del software. Se gli studenti non dispongono ancora di un account Microsoft per il servizio gestito (MSA) o sono parte di un'istanza Active Directory dell'università, verrà chiesto loro di creare un account corrispondente alla posta elettronica di ateneo. Gli studenti che sperimentano problemi di accesso all'hub di Education possono esaminare la guida alla risoluzione dei problemi disponibile di seguito o contattare il supporto studenti all'indirizzo [https://aka.ms/studentsupport](https://aka.ms/studentsupport).

## <a name="managing-access-for-students"></a>Gestione dell'accesso per gli studenti
È possibile gestire l'accesso per gli studenti in base al dominio usato dagli studenti stessi per registrarsi in Azure Dev Tools for Teaching. Accedere al [portale di gestione](https://azureforeducation.microsoft.com/en-us/account/Subscriptions) relativo ad Azure Dev Tools for Teaching. Nel portale verranno visualizzate tutte le sottoscrizioni. Scegliere la sottoscrizione per la quale si desidera gestire l'accesso.
