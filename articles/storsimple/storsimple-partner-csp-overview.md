# <a name="what-is-storsimple-for-cloud-solutions-providers-program"></a>Che cos'è StorSimple per il programma Cloud Solutions Provider?


## <a name="overview"></a>Panoramica

Microsoft Azure StorSimple è un approccio univoco all'archiviazione aziendale con funzionalità di archiviazione cloud realmente ibride. Il servizio permette inoltre ai clienti di sfruttare i vantaggi economici offerti dall'archiviazione cloud per i dati inattivi, mantenendo i dati critici in locale per assicurare livelli ottimali di prestazioni. 

L'array virtuale StorSimple per provider di soluzioni cloud (CSP) consente ai partner di sfruttare questa opportunità. I partner sono proprietari del ciclo di vita del cliente end-to-end con attività dirette di provisioning, fatturazione e supporto dei servizi cloud di Microsoft. In sintesi, i partner possono ora eseguire transazioni relative a StorSimple, insieme ai propri servizi, con i clienti.

Per altre informazioni su StorSimple per CSP, vedere [Azure CSP overview](https://docs.microsoft.com/en-us/azure/cloud-solution-provider/overview/azure-csp-overview) (Panoramica su Azure CSP).

Per altre informazioni su fatturazione, prezzi, incentivi e per ottenere assistenza per CSP, passare a [StorSimple per CSP: domande frequenti](storsimple-partner-csp-faq.md). 

## <a name="deploy-and-manage-storsimple-for-csp"></a>Distribuire e gestire StorSimple per CSP

StorSimple per CSP è disponibile come servizio basato sull'uso in tutti i mercati in cui StorSimple è disponibile attualmente. StorSimple per CSP usa il portale di Azure e il servizio Gestione dispositivi StorSimple. I partner CSP possono creare un servizio Gestione dispositivi StorSimple per gestire gli array virtuali, le condivisioni, i volumi e i backup di StorSimple. È possibile amministrare tutti gli array virtuali registrati per il servizio Gestione dispositivi StorSimple usando il portale di Azure.

Per altre informazioni, vedere [Distribuire e gestire StorSimple Virtual Array per CSP](storsimple-partner-csp-deploy.md).
