# <a name="windows-phone-silverlight-sdk-release-notes"></a>Note sulla versione dell'SDK per Windows Phone Silverlight
## <a name="330-04192016"></a>3.3.0 (19/04/2016)
Parte del pacchetto Nuget *MicrosoftAzure.MobileEngagement***v3.4.0**

* Aggiunta dell'API "TestLogLevel" per abilitare/disabilitare/filtrare i log della console generati da SDK.

## <a name="320-01222016"></a>3.2.0 (22/01/2016)
Parte del pacchetto Nuget *MicrosoftAzure.MobileEngagement***v3.3.0**

* Schemi personalizzati, ms-appx e ms-appdata, disponibili negli URL di azione.
* Miglioramenti della stabilità.

## <a name="310-11202015"></a>3.1.0 (20/11/2015)
Parte del pacchetto Nuget *MicrosoftAzure.MobileEngagement***v3.2.0**

* Arresto anomalo fisso durante la richiesta di id del dispositivo subito dopo l'inizializzazione.
* La console registra miglioramenti.

## <a name="300-04032015"></a>3.0.0 (03/04/2015)
* L'icona di notifica predefinita è stata aggiornata.
* È possibile inviare feedback relativo all'azione di notifica di sistema quando si fa clic su una notifica.
* È stata risolta una race condition relativa alla notifica di sistema.

## <a name="200-02172015"></a>2.0.0 (17/02/2015)
* Versione iniziale di Azure Mobile Engagement
* La configurazione di appId/sdkKey viene sostituita da una configurazione della stringa di connessione.
* Sono stati introdotti miglioramenti per la sicurezza.
