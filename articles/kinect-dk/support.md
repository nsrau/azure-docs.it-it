# <a name="azure-kinect-support-options-and-resources"></a>Opzioni e risorse per il supporto Kinect di Azure

In questo articolo vengono illustrate le diverse opzioni di supporto.

## <a name="community-support"></a>Supporto della community

Esistono diversi modi per rispondere alle proprie domande tramite i forum pubblici:

- [StackOverflow](https://stackoverflow.com/search?q=azurekinect&s=3b855ed0-8564-4961-856f-9614aeab4c0d&s=fd9ea920-622c-4d8e-b908-ec996e1f1403), in cui è possibile porre domande o eseguire ricerche nella libreria di domande esistente.
- [GitHub](https://github.com/Microsoft/Azure-Kinect-Sensor-SDK), in cui è possibile porre domande, aprire nuovi bug o contribuire allo sviluppo di Azure Kinect Sensor SDK.
- [Fornire commenti e suggerimenti](https://feedback.azure.com/forums/920053-azure-kinect-dk), in cui è possibile condividere le proprie idee per il futuro del prodotto e votare un'idea esistente.

## <a name="assisted-support"></a>Supporto assistito

Sono disponibili diversi modi per ottenere un supporto assistito per Kinect di Azure.

### <a name="microsoft-qa"></a>Domande e risposte Microsoft

Per ricevere risposte rapide e affidabili a domande di carattere tecnico sui prodotti, è possibile interagire con tecnici Microsoft, con MVP (Most Valuable Professional) di Azure o con la community di esperti nella pagina [Domande e risposte Microsoft](/answers/products/azure), la destinazione ideale di Azure per il supporto della community.

- [Microsoft Q&un per Kinect di Azure](/answers/topics/azure-kinect-dk.html), in cui è possibile porre domande o eseguire ricerche nella libreria di domande esistente.

### <a name="development-azure-kinect-on-azure"></a>Sviluppo di Kinect di Azure in Azure

I sottoscrittori di Azure possono creare e gestire le richieste di supporto nel portale di Azure. Il supporto dello sviluppo uno-a-uno per il body tracking, Sensor SDK, Speech Device SDK o servizi cognitivi di Azure è disponibile per i sottoscrittori di Azure con un [piano di supporto di Azure](https://azure.microsoft.com/support/plans/) associato alla propria sottoscrizione.

  - Un [piano di supporto di Azure](https://azure.microsoft.com/support/plans/) è associato alla sottoscrizione di Azure?  [Accedere a portale di Azure](https://ms.portal.azure.com/) per inviare un evento imprevisto.
  - Hai bisogno di una sottoscrizione di Azure? Le [Opzioni della sottoscrizione di Azure](https://azure.microsoft.com/pricing/purchase-options/) forniranno ulteriori informazioni sulle diverse opzioni.
  - Serve un piano di supporto? [Selezionare il piano di supporto](https://azure.microsoft.com/support/plans/)

### <a name="azure-kinect-on-premises-or-other-cloud-services"></a>Azure Kinect locale o altri servizi cloud

Per il supporto tecnico con Sensor SDK e body tracking SDK in locale, aprire un ticket per il supporto professionale nel [portale del supporto tecnico Microsoft](https://support.microsoft.com/supportforbusiness/productselection?sapId=c49ea5bb-2b09-8612-be35-d55159732667).

### <a name="azure-kinect-dk-device"></a>Dispositivo Azure Kinect DK

Prima di contattare il supporto hardware, assicurarsi di aver configurato e aggiornato l'aggiornamento di Azure Kinect DK. Per verificare se il dispositivo funziona, usare il [Visualizzatore Kinect di Azure](azure-kinect-viewer.md). Ulteriori informazioni sono disponibili nella pagina della [Guida di Azure Kinect dk](./index.yml) .
È anche possibile esaminare i [problemi noti e la risoluzione dei](troubleshooting.md)problemi.

[Ottenere assistenza](https://support.microsoft.com/supportforbusiness/productselection?sapId=f77b1b95-721e-43a0-2db8-b01e81a3f813) con funzionalità di dispositivo o sensore, aggiornamenti del firmware o opzioni di acquisto.

Per ulteriori informazioni sulle offerte di supporto, vedere il [supporto tecnico Microsoft per le aziende](https://support.microsoft.com/help/4341255/support-for-business).
